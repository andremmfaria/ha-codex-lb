# HA Codex LB

Home Assistant add-on repository for [Codex LB](https://github.com/Soju06/codex-lb) — an open-source load balancer for the OpenAI/ChatGPT API with OAuth token rotation, rate limiting, and sticky session routing.

## Installation

1. In Home Assistant, go to **Settings** → **Add-ons** → **Add-on Store**.
2. Click the three dots (⋮) → **Repositories**.
3. Add: `https://github.com/andremmfaria/ha-codex-lb`
4. Find **Codex LB** and install.
